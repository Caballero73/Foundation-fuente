Alternar una clase de CSS

  <p><a class="button large" data-toggle="menuBar">Expandir!</a></p>

  <ul class="menu" id="menuBar" data-toggler=".expanded">
    <li><a href="#">Uno</a></li>
    <li><a href="#">Dos</a></li>
    <li><a href="#">Tres</a></li>
    <li><a href="#">Cuatro</a></li>
  </ul>

El valor de data-toggler es la clase que desea alternar. También dale al elemento una ID única para que pueda ser el objetivo.

Luego, agregue data-toggle a cualquier elemento, con el ID del objetivo como el valor del atributo. Ahora, cada vez que haga clic en este elemento, la clase activará y desactivará el objetivo.

Usar con HTML

  <img data-interchange="[assets/img/int-small.jpg, small], 
  [assets/img/int-medium.jpg, medium], [assets/img/int-large.jpg, large]">

También puede intercambiar fragmentos enteros de HTML. Esto le permite cargar componentes amigables para dispositivos móviles en pantallas pequeñas o versiones más avanzadas en pantallas grandes.

En el siguiente ejemplo, hemos aplicado data-interchange a un elemento <div> lugar de a un <img> , y las rutas son a archivos HTML en lugar de imágenes.



Usar con HTML

  <div class="bg-int" data-interchange="[assets/img/bg-small.jpg, small], [assets/img/bg-medium.jpg, medium], [assets/img/bg-large.jpg, large]">
    <div>
      "La violencia es el último recurso del incompetente".
      <cite>Salvor Hardin</cite>
      <hr>
      <small>Texto sobre un DIV cuyo background es intercambiable</small>
      <br>
    </div> 
  </div>

Al usar Interchange en un elemento que no sea <img> , puede pasar una ruta de imagen en lugar de una ruta HTML, y la propiedad background-image del elemento se establecerá en la ruta de la media query correspondiente.
